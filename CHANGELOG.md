## [0.2.0] - 2026-04-24

### Added

- Unit test suite (33 tests) covering `rules.py`, `cascade.py`,
  `metrics.py`, and `monthly_signal.py`. Tests run in under a second
  and do not access the network.
- GitHub Actions workflow `.github/workflows/test.yml`, triggered on
  push and pull request to `main` or `public`. Runs `pytest` and
  `ruff check`. Must be green before merge.
- `docs/methodologie.md` — a 1500-word methodology page explaining
  why the strategy uses two orthogonal signals, how each is built,
  and when each has mattered on historical BTC data. Includes seven
  bibliographic references (DOIs verified) and an AI-in-writing
  disclosure paragraph.
- Three annotated historical figures in `docs/assets/methodology/`
  (`signaux_history.png`, `powerlaw_band.png`,
  `cascade_position_history.png`), generated by the new
  `chillbtc-methodology-figures` CLI entry point.
- Navigation pointers to the methodology page from the auto-generated
  `docs/signaux.md` header, the `README.md`, and the
  `docs/index.html` landing page.
- Ruff lint configuration in `engine/pyproject.toml`
  (`[tool.ruff]` section).
- `CHANGELOG.md` (this file).

### Changed

- Public-facing documentation renames the two signals from the
  internal codes `R1 / R3` to `tendance / valorisation`. Python
  identifiers (`signal_r1`, `signal_r3`, `N_TSMOM`) and CSV column
  names are unchanged.
- `CONTRIBUTING.md` now documents the test commands, the test CI
  workflow, and the lint rule.

## [0.1.0] - 2026-04-23

### Added

- Initial public release of the ChillBTC monthly Bitcoin strategy.
- Frozen strategy live: TSMOM 11-month tendance signal, Power Law
  valorisation signal (`k_low = 0.6`, `k_high = 2.5`, `A = -16.917`,
  `N = 5.8`), cascade convention `strict_r1_def`, three-step dosage
  100 / 50 / 0.
- Auto-updating pages on GitHub: `docs/signaux.md`,
  `docs/historique-annuel.md`, `docs/historique-mensuel.md`.
- Monthly workflow `.github/workflows/monthly-update.yml`, scheduled
  on the 1st of each month at 06:00 UTC.
- Landing page `docs/index.html` with brand assets.
- `LICENSE` (CC BY-NC-SA 4.0) and `CONTRIBUTING.md`.
